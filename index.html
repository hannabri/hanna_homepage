Projects


Predictive Text Input Final bachelor project in Computational Linguistics. This application suggests words while typing a message. It is composted of
  • prediction where the user did not start typing a new word yet and
  • completion where at least the first character is already known.
For word completion, we used a trie structure as it allows to store letters that compose words. This makes it easier to extract
the list of suggestions at each character written by the user. The suggestions are based on probabilities calculated over a French SMS corpus.
Here is the link to the GitHub project.
Semi Supervised Word Sense Disambiguation Final M1 project in Computational Linguistics. This project compares two approaches for word sense disambiguation:
  • a classification approach where training is supervised and
  • a clustering approach where training is not supervised.
We also tested a seeded clustering approach where the centroids were initiated with up to 5 examples. The training data
was taken from a French verb dataset which includes around 50 examples per verb.
In our project, the seeded clustering worked best of all tested methods.
Here is the link to the GitHub project.
Fine-tune mBERT University project to analyze mBERT's generalization capacities. The model was fine-tuned five times on five different languages:
  • French
  • Spanish
  • German
  • Turkish
  • Czech
Each fine-tuned model was tested on all languages. For the language the model was fine-tuned on, it achieved more than 90 % accuracy.
When used with another language, the models performance decreases drastically. Only the Czech model used with German performs well
with more than 75 % accuracy and vice versa, the model trained on German und used with Czech achieves around 60 % accuracy.
Here is the link to the GitHub project.

Professional Experiences

Internship NLP / GenAI - Dassault Systèmes

Six-month internship between January and July 2025 to build a recommandation system. The steps followed during the internship were the following:

  • Dynamic Web Scraping to build a test dataset with data from the web.
  • Creation of an RDF graph written in Turtle syntax.
  • Creation of a Retrieval Augmented Generation (RAG) system using a simple Re-Act agent and querying two different data bases:
    • the created RDF graph for structured data and
    • a FAISS vector database for unstructured data.
  • SPARQL query generation with a Mistral LLM, a Chain-of-Thought prompt and in-context learning similar to the user question.
  • Creation of a simple user interface allowing an easier way to ask questions.
After completing the RAG system, I worked with an internal Business Intelligence tool for data analysis.
